# 🏪 مؤسسة عقيق الشرق للتجارة - سباكة وكهرباء ومواد بناء

هذا المستودع يحتوي على الكود المصدري للموقع الإلكتروني الخاص بـ **مؤسسة عقيق الشرق للتجارة**، وهو متجر متخصص في بيع مستلزمات السباكة، الكهرباء، ومواد البناء في مدينة الرياض. تم تصميم وتطوير الموقع ليكون واجهة رقمية احترافية، سريعة، ومحسنة بالكامل لمحركات البحث (SEO).

## 🚀 المميزات (Features)

- **أداء فائق وسرعة تحميل (High Performance):** مبني بالكامل باستخدام HTML, CSS, Vanilla JavaScript بدون أي مكتبات خارجية ثقيلة، مما يضمن تقييمات مرتفعة جداً في Google Lighthouse.
- **تصميم متجاوب بالكامل (Fully Responsive):** واجهة مستخدم (UI/UX) مرنة تتكيف بسلاسة مع جميع الشاشات (أجهزة مكتبية، أجهزة لوحية، وهواتف ذكية).
- **تحسين محركات البحث (Technical SEO):**
  - علامات وصفية (Meta Tags) وكلمات مفتاحية مدروسة.
  - دعم كامل لـ Open Graph و Twitter Cards لضمان ظهور مميز عند مشاركة الرابط على منصات التواصل الاجتماعي.
  - تطبيق البيانات المهيكلة (Schema Markup / JSON-LD) مخصصة كـ `HardwareStore` و `LocalBusiness` لدعم الـ Local SEO.
- **تجربة مستخدم تفاعلية (Interactive UX):** 
  - قائمة تنقل جانبية للموبايل.
  - قسم أسئلة شائعة (FAQ) تفاعلي (Accordion).
  - أزرار عائمة (Floating Actions) للوصول السريع.
- **تكامل مباشر مع واتساب (WhatsApp Integration):** نموذج تواصل ذكي يقوم بجمع بيانات العميل (الاسم، الجوال، نوع الاستفسار، والتفاصيل) وتوجيهها مباشرة كرسالة منسقة إلى واتساب المبيعات الخاص بالمؤسسة.

## 🛠️ التقنيات المستخدمة (Technologies)

- **HTML5:** بناء هيكل دلالي (Semantic Structure) سليم.
- **CSS3:** تنسيقات حديثة باستخدام Flexbox و CSS Grid ومتغيرات الألوان (CSS Variables).
- **JavaScript:** سكريبتات خفيفة للتحكم بالتفاعلات ونموذج الواتساب.
- **Google Fonts:** استخدام خط "Cairo" لدعم الطباعة العربية بشكل عصري وواضح.
